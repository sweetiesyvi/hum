## Student Manager

Student Manager is a simple full stack web application that allows users to manage a list of students.

## User Story

As an individual developer  
I want to build a full stack web application with CRUD functionality  
So that I can demonstrate a complete data round trip between the browser, server, and database.

## User Experience

The application is designed to be simple and easy to use.

Users can:
- Add a new student
- View all students
- Update a student name
- Delete a student

All actions are performed instantly and stored in the database, allowing the data to persist even after refreshing the page.

## Technologies Used

Frontend:
- HTML
- CSS
- Bootstrap 5
- jQuery
- normalize.css

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Development Tools:
- dotenv
- nodemon
- Git and GitHub

## Features

The application demonstrates a complete CRUD system:

Create – add a new student  
Read – display the list of students  
Update – edit a student's name  
Delete – remove a student from the database

This shows a full stack data round trip between the client, server, and database.
